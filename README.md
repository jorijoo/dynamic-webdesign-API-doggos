# API interface for https://dog.ceo/dog-api/

View and filter dog breeds from the dog.ceo open source dog image database.
View random images from database or filtered by breed and sub-breed.

## Requirements

- npm
- react

## Installation

$ npm init react-app .

## Usage

$ npm start
